# 🏘️ Ames Housing Price Predictor

This project provides a robust machine learning solution for predicting house prices in Ames, Iowa. 

## Features
* **Custom Scikit-Learn Pipelines:** Robust preprocessing for numerical and categorical data.
* **Feature Engineering:** Automated calculation of house age, total area, and room efficiency.
* **Advanced Imputation:** Using MICE (IterativeImputer) for missing values..
* **Deployment:** Interactive Web App built with Streamlit.

## Installation & Usage
1. Clone the repo: `git clone https://github.com/IbrahimElmansoury/Ames-Housing-Price-Predictor.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
